Image Restoration Evaluation

In this study, we are evaluating different AI methods for image restoration. Here's what you need to know:

  • The reference image (shown on the left) is the person's original high-resolution real image.
  • We have a very low-resolution version of this image that has been restored using nine different AI methods.
  • Your task is to rank these restored images based on their similarity to the reference image.

How to Rank:

  1. Click on the image that looks most similar to the reference image first (this will be ranked as 1).
  2. Continue clicking on images in order of similarity, with the least similar image being ranked as 9.
  3. If you want to change your ranking, simply click on an image again to deselect it.

Comparing Images:

  • Use the slider below each image to compare it with the reference image.
  • Slide left to see the reference image, slide right to see the restored image.
  • This can help you make more accurate comparisons.

Note: There are 20 pages in total. Please complete all pages for a comprehensive evaluation.

Thank you for your participation in this important research!
